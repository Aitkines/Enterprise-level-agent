# 系统使用说明（最新版）

## 1. 访问方式

有两种方式进入系统：

1. 打开 `1_作品与答辩材料/运行网址.txt` 中的网址。
2. 扫描 `1_作品与答辩材料/系统访问二维码.png`。

当前固定域名网址：

`https://aroma-espresso-patience.ngrok-free.dev`

## 2. 首次访问提示

若浏览器先显示 ngrok 提示页，点击 `Visit Site` 即可进入系统。

## 3. 主要功能

1. 对话分析：输入问题后生成分析结果与结构化输出。
2. 财务分析：查看目标公司财务数据与关键指标。
3. 对标比较：查看与同类对象的对比图表与结论。
4. 报告导出：支持报告页面与 PDF 导出。

## 4. 建议演示流程

1. 打开系统首页。
2. 在对话区输入一个分析问题。
3. 展示财务与对标页面。
4. 最后演示报告导出。

## 5. 常见问题

1. 页面打不开：
检查后端和 ngrok 是否正在运行。

2. 地址变了：
说明未使用固定域名启动，请按部署说明使用 `-NgrokDomain` 启动。

3. 二维码失效：
重新执行启动脚本后会自动更新二维码。
