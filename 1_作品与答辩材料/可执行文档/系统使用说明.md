# 光之耀面系统使用说明

## 1. 系统简介

光之耀面（Radiant Surface）是一套面向企业研究、财务分析和智能问答场景的交互式系统。系统围绕企业研究过程中的信息获取、数据分析、对比判断和报告生成等需求进行设计，支持用户以自然语言完成研究任务。

## 2. 系统主要功能

### 2.1 多轮对话

用户可以通过自然语言输入公司名称、行业问题或研究需求，系统支持连续对话，并可围绕上下文持续展开分析。

### 2.2 会话管理

系统支持创建、查看、重命名和删除会话，便于用户对不同研究主题进行归档和管理。

### 2.3 财务分析

用户输入目标公司后，系统可返回相关财务数据、结构化指标和图表，用于辅助经营分析和投研判断。

### 2.4 公司对比分析

系统支持对目标公司与同行或可比公司进行横向对比，输出关键指标差异和分析结论。

### 2.5 研究报告生成

系统可根据当前会话内容生成研究报告，并支持导出为 PDF 文档，便于展示和归档。

## 3. 使用流程

### 3.1 启动系统

在项目目录中运行：

```powershell
.\start_dev.ps1
```

如为首次使用，可运行：

```powershell
.\start_dev.ps1 -InstallDeps
```

启动完成后，在浏览器中访问：

`http://127.0.0.1:5173`

### 3.2 进入系统

打开网页后，用户可进入系统主界面，界面包含会话区、功能导航区和内容展示区。

### 3.3 发起研究问题

用户可输入以下类型的问题：

- 某家公司的经营状况如何
- 某家公司的财务指标表现如何
- 某家公司与同行相比有哪些优势和风险
- 请根据当前对话生成研究报告

### 3.4 查看结果

系统将返回文字分析结果，并在适用场景下展示图表、结构化数据和报告内容。

## 4. 典型使用场景

### 场景一：公司研究

输入目标公司名称或研究问题，系统可给出业务概况、风险提示和分析结论。

### 场景二：财务指标分析

输入公司名称后，可查看收入、利润、毛利率等财务指标变化情况。

### 场景三：公司对比

通过输入目标公司或股票代码，系统可生成与同行公司的对比结果，并展示差异分析。

### 场景四：报告导出

在完成对话研究后，可生成报告内容并导出 PDF 文件。

## 5. 使用注意事项

1. 系统部分能力依赖外部模型接口和数据接口，使用前需确保相关密钥已正确配置。
2. 若网络不稳定或外部接口异常，可能影响回答速度和结果生成。
3. 在进行正式展示前，建议提前准备示例问题，保证演示流程顺畅。
4. 若用于线上部署，应确保前端与后端地址配置正确。

## 6. 建议展示方式

在比赛答辩或作品演示时，建议按照以下顺序展示：

1. 启动系统并进入首页
2. 演示多轮对话功能
3. 演示财务分析功能
4. 演示公司对比功能
5. 演示报告生成与 PDF 导出功能

通过上述流程，可完整体现系统在企业研究与投研分析场景中的应用价值。
